# Knowledge Technology Practical

**Welcome to our project** 

This repository will contain our knowledge base and models.

**Prerequisites**

```
pip install -r requirements.txt
```
This script will let you download all the dependencies.


**How to run the Streamlit app**

Open a terminal in the working directory. Run the command **streamlit run main.py** to see the base interface.


**Structure**

```bash
├───main.py  # main file to run.
├───.gitignore
├───README.md
├───requirements.txt
```
